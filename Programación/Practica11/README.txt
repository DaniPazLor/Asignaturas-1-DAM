#####################################MANUAL DE USUARIO PARA APLICACIÓN DE NOTARÍA#################################################

1.- Ejecutamos la aplicación y Se mostrará un menú:
	
	1/ Insertar datos cliente
	2/ Insertar datos escritura
	3/ Mostrar datos clientes
	4/ Mostrar datos escrituras
	5/ Modificar datos de cliente
	6/ Mostrar nombre cliente con escritura 'CPVE'
	7/ Salir de la aplicación
	
2.- Se mostrará un menú donde podemos seleccionar cualquiera de las 7 opciones

3.- Si seleccionamos la primera o la segunda la aplicación nos va a solicitar esos datos para insertarlos en la BBDD

4.- Si seleccionamos la tercera o la cuarta mostrará todos los datos contenidos en las tabla clientes o escrituras

5.- En caso de seleccionar la opción 5 nos va a solicitar el código de cliente que queremos modifcar y a continuación 
nos va a solicitar el nuevo nombre y teléfono cliente que se sobreescribirá al anterior

6.- Seleccionando la opción 6 mostrará una tabla que contiene los nombres de clientes con escrituras 'CPVE'

7.- La séptima y última opción nos sacará del programa mostrando un mensaje de despedida
